# Basic Shell
**About**

This program acts as a mini Linux shell.

**Scope**
- It supports built-in shell commands like "cd" and "exit".
- It supports input and output redirection.
- It supports background process.

**Usage**
```sh
$ ./basicshell.o
```

**Requirements**
- Linux OS
- gcc
